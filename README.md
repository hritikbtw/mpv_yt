# mpv-ytsearch

Search and play YouTube videos inside mpv using the native selection UI.

## Keybinds
* `Alt+y` - Quick search (replaces playback)
* `Alt+Y` - Deep search (replaces playback)
* `Alt+a` - Append to playlist

## Installation
Place `youtube-search.lua` and `ytbackend.py` in `~/.config/mpv/scripts/`.

## Requirements
* Python 3
* mpv
