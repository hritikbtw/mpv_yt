## Keybinds
* `Alt+y` - Quick search, faster, less results
* `Alt+Y` - Deep search, takes more time, ~50 results (configurable)
* `Alt+a` - Append to playlist

## Installation
Place `youtube-search.lua` and `ytbackend.py` in `~/.config/mpv/scripts/`.

## Requirements
* yt-dlp (not used for search, but for playing)
* mpv
